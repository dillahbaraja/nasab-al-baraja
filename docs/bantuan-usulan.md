# Bantuan Pengguna: Cara Mengajukan Usulan

Dokumen ini ditujukan untuk pengunjung website yang ingin:

1. Mengajukan usulan tambah anak
2. Mengajukan usulan perubahan nama

Catatan:
- Anda tidak perlu login admin untuk mengajukan usulan.
- Semua usulan akan tampil langsung di tree dengan status pending.
- Usulan baru akan menjadi data final setelah diverifikasi admin.

---

## 1. Membuka popup person

1. Buka website Nasab Al-Baraja.
2. Cari person yang ingin Anda usulkan perubahannya.
3. Klik person tersebut untuk membuka window popup detail.

Screenshot yang perlu ditambahkan:
- Tampilan tree
- Contoh person/node yang dipilih
- Popup detail person yang sudah terbuka

---

## 2. Usulan tambah anak

1. Di dalam popup person, cari tombol `Usulan tambah anak`.
2. Klik tombol tersebut.
3. Akan muncul form isian:
   - Nama Arab
   - Nama Inggris / Latin
4. Isi nama anak yang ingin diusulkan.
5. Jika ingin menambahkan lebih dari satu anak, klik tombol `+`.
6. Setelah selesai, klik tombol `Simpan usulan`.

Hasil setelah disimpan:
- Usulan anak langsung tampil di tree
- Node usulan ditandai sebagai pending
- Akan muncul penanda `Menunggu verifikasi admin`

Screenshot yang perlu ditambahkan:
- Tombol `Usulan tambah anak`
- Form tambah anak
- Tombol `+` untuk tambah baris anak
- Tampilan node pending setelah usulan disimpan

---

## 3. Usulan perubahan nama

1. Di dalam popup person, klik tombol `Usulan perubahan nama`.
2. Form perubahan nama akan muncul.
3. Nama lama akan otomatis terisi pada field yang tersedia.
4. Ubah nama Arab dan/atau nama Inggris sesuai usulan Anda.
5. Klik `Simpan usulan`.

Hasil setelah disimpan:
- Nama usulan langsung tampil ke publik
- Data akan diberi penanda pending
- Akan muncul status `Menunggu verifikasi admin`

Screenshot yang perlu ditambahkan:
- Tombol `Usulan perubahan nama`
- Form perubahan nama dengan isi otomatis
- Tampilan nama pending setelah usulan disimpan

---

## 4. Tanda bahwa usulan masih pending

Selama usulan belum diverifikasi admin:
- Node/person terkait akan diberi penanda visual merah
- Akan tampil tulisan `Menunggu verifikasi admin`
- Data masih dapat diedit atau dibatalkan

Screenshot yang perlu ditambahkan:
- Contoh node pending di tree
- Contoh status pending di popup person

---

## 5. Mengedit usulan yang masih pending

Jika usulan masih pending:

1. Buka kembali popup person terkait.
2. Klik tombol `Edit usulan`.
3. Perbaiki data yang diinginkan.
4. Klik `Simpan perubahan usulan`.

Screenshot yang perlu ditambahkan:
- Tombol `Edit usulan`
- Form edit usulan

---

## 6. Membatalkan usulan

Jika Anda ingin membatalkan usulan:

1. Buka popup person terkait.
2. Klik tombol `Batalkan usulan`.
3. Konfirmasi pembatalan.

Hasil:
- Usulan akan dihapus dari tampilan publik
- Node pending tidak akan tampil lagi

Screenshot yang perlu ditambahkan:
- Tombol `Batalkan usulan`
- Dialog konfirmasi pembatalan

---

## 7. Apa yang dilakukan admin

Setelah usulan masuk:
- Admin dapat melihat person yang masih pending
- Admin bisa memilih:
  - Setujui usulan
  - Tolak usulan
  - Skip ke verifikasi berikutnya

Jika admin menyetujui:
- Penanda pending dihapus
- Data menjadi data final

Jika admin menolak:
- Usulan dihapus dari tampilan publik

Screenshot yang perlu ditambahkan:
- Tampilan popup admin pada node pending
- Tombol `Setujui usulan`
- Tombol `Tolak usulan`
- Tombol `Skip`

---

## 8. Notice / notifikasi

Saat ada usulan baru:
- Notifikasi akan muncul di Notice
- Usulan terbaru tampil di urutan paling atas
- Notice akan tetap muncul selama statusnya masih pending
- Jika usulan diverifikasi atau dibatalkan, notice akan hilang otomatis

Screenshot yang perlu ditambahkan:
- Daftar notice
- Contoh notice usulan tambah anak
- Contoh notice usulan perubahan nama

---

## 9. Ringkasan singkat

- Pengunjung biasa dapat mengajukan usulan tanpa login admin
- Usulan langsung terlihat publik, tetapi belum final
- Semua usulan pending menunggu verifikasi admin
- Usulan bisa diedit atau dibatalkan selama masih pending
- Admin menentukan apakah usulan disetujui atau ditolak
